まずはデグレが検知出来るようにテストがあるのか、既存のテストが動作が変わっていないことを担保出来るのかを確認してください

もしデグレ検知ができないと判断した場合は、テストコードの追加や修正を行ってください。テストコードは、変更されたファイルに関連する機能やロジックをカバーするようにしてください。

指定されたファイルに対してレビューツールを使ってレビューします
```
cat <file> | swiss review file-refactor
```

指摘内容に従ってリファクタリングを行います
修正は指定されたファイルに可能な限り閉じてください

またレビュワーはコードをすべて読んでおらず、過去の指摘も覚えていないため、矛盾した指摘をする可能性があります
その場合は私に判断を仰いでください

レビュツールのレビュが通る、もしくは私の判断が必要になるまでリファクタリングを繰り返してください
